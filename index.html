Map of the world

Click and drag the map or use the zoom controls and navigate to a location of your choosing.
Click on any location to view the current weather data!

For this project, I built an application to meet the following functionality: displays a map of the world which a user can move and zoom.
When a user clicks on a given location, the webapp will display the current weather data for that location.

For the map, I integrated Mapbox's API, and for the weather data I integrated Open Weather Map's data.

This webapp was designed and built by Hannah R. McMahon. April 2019.
